Задание 4

1)Составьте GET-запрос к серверу по адресу http://sandbox.usm.com, указав в заголовке User-Agent ваше имя и фамилию.

    GET / HTTP/1.1
    Host: sandbox.usm.com
    User-Agent: Anton Salkutan
    
1.2)Что такое User-Agent и для чего он используется?
User-Agent — это информация о клиенте (браузере или пользователе), которая отправляется серверу

2.)Составьте POST-запрос к серверу

    POST /cars HTTP/1.1
    Host: sandbox.usm.com
    Content-Type: application/x-www-form-urlencoded
    make=Toyota&model=Corolla&year=2020

2.1)Какие еще методы HTTP-запросов существуют и для чего они используются?

    GET — получить данные
    POST — отправить данные
    PUT — обновить данные
    DELETE — удалить данные

3Составьте PUT-запрос к серверу

    PUT /cars/1 HTTP/1.1
    Host: sandbox.usm.com
    User-Agent: Anton Salkutan
    Content-Type: application/json

    {
     "make": "Toyota",
     "model": "Corolla",
     "year": 2021
    }

3.1)В чем разница между PATCH и PUT запросами?

PUT обновляет весь объект, PATCH — только часть данных
